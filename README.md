## Ön kurulum
Visual Studio Code kullanmanızı tavsiye ederim.

terminal'linizi açtıktan sonra şu kodları yazınız


    git clone https://github.com/ofturkey0/pythontextbasedgame
    
ardından

    cd pythontextbasedgame
dosya konumuna girip

    code .
